# 🔒 EthicalHacking Academy - TINKODE

O platformă completă de învățare a Ethical Hacking în limba română.

## 📋 Caracteristici

✅ **Lecții Interactive** - Cursuri complete despre ethical hacking  
✅ **Quiz-uri** - Teste interactive pentru a verifica cunoștințele  
✅ **Autentificare** - Sistem de login/register pentru utilizatori  
✅ **Dashboard Personalizat** - Tracker de progres pentru fiecare utilizator  
✅ **Design Profesional** - Interfață modernă și responsivă  
✅ **Branding TINKODE** - Platformă creată de TINKODE  

## 🚀 Cum să pornești

1. Clone repo-ul
   ```bash
   git clone https://github.com/ionutivan21/Hacking-web.git
   cd Hacking-web
   ```

2. Deschide `index.html` în browser

## 📁 Structura Proiectului

```
Hacking-web/
```
